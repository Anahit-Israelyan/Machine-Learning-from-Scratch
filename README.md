# Machine Learning from Scratch

This repository contains Python implementations of foundational Machine Learning algorithms built entirely from scratch. 

The goal of this project is to deepen my mathematical and algorithmic understanding of these models by implementing their core mechanics without relying on high-level frameworks like Scikit-learn.

Currently, this repository includes scratch implementations of:

* **Decision Tree Classifier:** Implementation of node splitting, entropy/Gini impurity calculations, and tree traversal.
